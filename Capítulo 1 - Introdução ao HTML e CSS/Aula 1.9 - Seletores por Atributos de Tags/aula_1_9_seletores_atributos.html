Seletores por atributos

  • [atributo] - Seleciona todos os elementos que contenham este atributo
  • [atributo=valor] - Seleciona todos os elementos que contenham o atributo que tenha o valor correspondente
  • [atributo^=valor] - Seleciona todos os elementos que tenham o atributo que o valor inicie com "valor"
  • [atributo$=valor] - Seleciona todos os elementos que tenham o atributo que o valor termine com "valor"
  • [atributo*=valor] - Seleciona todos os elementos que tenham o atributo que contenha "valor" em qualquer lugar
